# File Integrity Monitor

## Description
A Python-based cybersecurity tool used to monitor file integrity using hashing techniques.

## Features
## Features

- GUI-based File Integrity Monitoring
- SHA-256 hashing
- Real-time file monitoring
- Unauthorized modification detection
- Popup security alerts
- Log file generation
- Old and new hash comparison

## Technologies Used
- Python
- hashlib
- os

## How to Run

```bash
python file_integrity_monitor.py
```
