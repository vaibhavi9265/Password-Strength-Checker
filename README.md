# Password Strength Checker

## Domain
Cyber Security & Ethical Hacking

## Description
A GUI-based Password Strength Checker developed using Python and Tkinter.

## Features
- Weak/Medium/Strong password detection
- Progress bar
- Show/Hide password
- Password improvement suggestions
